# Exoplanet-Radius-Prediction-ML

Este proyecto utiliza **Machine Learning** para predecir el tamaño de planetas fuera de nuestro sistema solar basándose en su masa y temperatura.

## Resultados
* **Modelo Inicial (Lineal):** R² = 0.14
* **Modelo Final (Random Forest):** R² = 0.74
* **Error Medio:** 1.7 radios terrestres.

## Tecnologías
* Python (Pandas, Scikit-Learn, Matplotlib)
* Datos reales del NASA Exoplanet Archive.
